厌倦了枯燥的鼠标操作？想要为你的电脑桌面增添一点趣味和惊喜？
那就来试试 “指针蜗牛” 吧！
开启一只蜗牛追杀你的指针！体验畅快淋漓的热血江湖！

🌟 功能简介：
开启插件后，一只可爱的蜗牛将会出现在你的屏幕上，并执着地追踪你的鼠标指针。它缓缓爬行、坚持不懈，仿佛在进行一场永不停歇的追逐战！

🖱️ 鼠标一动，蜗牛就追！
🐌 可爱画风 + 轻松特效，缓解工作压力
⚙️ 简单易用，一键开启或关闭
👐 自由拖拽，你可以重新安排它的起点 （mac暂不支持）

🤭 奇怪的技能：

⏰️ 蜗牛分身（就是定时）：按住【ctrl】+【鼠标拖动】蜗牛，出现蜗牛分身后即可开启定时。蜗牛会沿着直线前进，使用指定时间到达指定位置。（mac暂不支持）
